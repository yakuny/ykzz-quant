## Why

A股可转债投资缺乏一个面向个人投资者的、可运行、可回测的策略研究工具。现有工具要么是黑盒预测（不可解释），要么是纯数据展示（无策略筛选和回测能力）。可转债投资的核心难点在于条款、流动性、风险事件和轮动规则，先把"规则引擎 + 数据 + 风险过滤 + 回测"做扎实，比上复杂模型更有价值。

## What Changes

- 新增数据采集层：基于集思录（AkShare `bond_cb_jsl`）获取全市场可转债实时快照，基于 Tushare 补充历史行情和基本信息
- 新增因子计算层：计算并落库转股价值、转股溢价率、双低值、剩余期限、到期收益率（YTM）
- 新增策略筛选引擎：实现双低策略和低溢价策略的筛选、排序、组合构建
- 新增风险过滤模块：过滤剩余规模过小、成交额过低、强赎高风险、正股ST、临近到期等标的
- 新增回测引擎：基于 pandas 向量化实现周频调仓回测，输出累计收益、年化收益、最大回撤、夏普比率、胜率、换手率
- 新增 FastAPI 接口层：提供策略推荐、单债详情、回测结果等查询接口
- 新增 Streamlit 展示层：今日策略推荐、排名、风险预警名单、单债详情页

## Capabilities

### New Capabilities

- `data-layer`: 数据采集、清洗与落库能力，包括集思录实时快照、Tushare历史行情、正股行情、条款事件
- `factor-layer`: 关键因子计算与每日落库，包括转股价值、转股溢价率、双低值、剩余期限、YTM
- `strategy-layer`: 双低策略和低溢价策略的筛选、排序、过滤与组合构建
- `backtest-layer`: 周频调仓的向量化回测引擎，含手续费、滑点、停牌处理、绩效统计
- `api-layer`: FastAPI 接口，提供策略推荐、单债详情、回测结果查询
- `ui-layer`: Streamlit 展示页，包括今日推荐、排名、风险预警、单债详情

## Impact

- 新增 Python 项目结构（app/ 目录），含数据、因子、策略、回测、API、展示等模块
- 依赖：akshare、tushare、fastapi、sqlalchemy、pandas、streamlit、apscheduler
- 数据库：SQLite（MVP），6 张核心表 + 1 张转股价调整历史表
- 日更任务脚本：每日自动采集 → 因子计算 → 策略运行
